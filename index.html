Quem sou

Sou desenvolvedor indie focado em game design — na criação de sistemas, mecânicas e experiências que fazem sentido para quem joga. Trabalho com Unity e Godot para dar vida às ideias.

Do lado técnico, uso Python para análise de dados e automação, e JavaScript para ferramentas web. Gosto de cruzar essas áreas: dados informando design, e design humanizando dados.

Acredito que bons jogos nascem de iteração constante, de ouvir o feedback e de nunca parar de testar.
